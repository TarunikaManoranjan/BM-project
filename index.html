Biomaterials Project – Reflex Action Demonstration

Using Arduino Uno, LM35 Sensor & Servo Motor

Introduction

This project simulates a human reflex action using a microcontroller. We created a working model using Arduino Uno, LM35 temperature sensor, and a servo motor to represent the muscle response. The setup mimics how biological systems react instantly to stimuli, integrating electronics with biomaterial concepts.

Theory & Principle

  • Reflex action is an involuntary and nearly instantaneous movement in response to a stimulus.
  • LM35 acts as a temperature stimulus detector, mimicking external environmental input.
  • Arduino Uno processes the input and simulates the central nervous system.
  • Servo motor represents the muscle movement reacting to the stimulus.
  • Threshold logic was implemented: if temperature crosses a set value, servo motor activates.

Conclusion

The project successfully demonstrated a reflex-like action using basic biomimetic principles. This can serve as a foundational model for more advanced prosthetics and biomedical automation.
